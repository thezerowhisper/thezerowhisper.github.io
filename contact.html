?Frequently Asked Questions

Are these calculators free to use?
Yes — completely free, forever. No login, no subscription, no paywall. All tools work in the browser without any sign-up.
Can I use these in a clinical setting?
These tools are designed as decision-support aids for qualified healthcare professionals. They should supplement, not replace, clinical judgment. Always verify calculations independently before acting on them. See our full Medical Disclaimer.
Is patient data stored anywhere?
No. All calculations happen locally in your browser. Nothing is transmitted or stored. See our Privacy Policy for full details.
Which guidelines do you follow?
Indian guidelines (NCDC, IAP, FOGSI, Jindal spirometry reference values) and international standards (WHO, NICE, BTS, CKD-EPI 2021, AKIN/KDIGO, AAP, ADA, RSSDI). Each calculator page lists its specific references.
How do I request a new calculator?
Use the "Suggest a Calculator" card above to open a GitHub issue, or fill out the contact form on this page. Include the tool name, the guideline or reference you want it built on, and a brief description of the use case.
I found a clinical error — what should I do?
Please report it immediately using the "Clinical Accuracy" GitHub issue link above, or email us at info@rxmedcalc.com. Include the calculator name, the value you believe is wrong, and the guideline or source supporting the correction. We investigate and correct promptly.
